# Exploratory Research — Fixed-Point Negation (FPN)

**Status: preliminary, unpublished, not peer-reviewed.**

This directory contains self-conducted research exploring a hypothesized
failure mode in AI self-modification ("Fixed-Point Negation"), including
theoretical framing and small-scale empirical trials.

## Known limitations (identified on review)

- Empirical trials use small sample sizes (n=3–5 per condition in most
  studies); several papers report results not statistically significant
  at p<0.05 (e.g. p=0.125, p=0.08).
- The core theoretical paper (`RT-2026-third-fundamental-limit.md`) states
  axioms without independent derivation; review identified circular
  reasoning in the main convergence theorem (Theorem D's proof depends on
  an assumption Theorem E has not independently established).
- Referenced code repositories and DOIs are listed as pending; no external
  reproduction has occurred.
- `RT-2026-E005` found the effect can be eliminated entirely by changing
  prompt framing (0% vs 100% incidence) — in tension with the "fundamental
  limit" framing used elsewhere in this directory.

This directory is retained as a research log, not a validated result. Read
each file's own limitations section before drawing conclusions.
